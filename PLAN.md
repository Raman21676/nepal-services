## ⚖️ LEGAL ANALYSIS & COMPLIANCE

### What is SAFE to Collect/Publish ✅
| Data Type | Legal Status | Source |
|-----------|--------------|--------|
| Hospital/Clinic Name | ✅ Safe | Public registration |
| Address | ✅ Safe | Public information |
| Phone numbers (landline/office) | ✅ Safe | Business contact |
| General location/ward | ✅ Safe | Public record |
| Services offered | ✅ Safe | Public information |
| Operating hours | ✅ Safe | Public information |
| Emergency availability | ✅ Safe | Public service info |
| Number of beds | ✅ Safe | Public record |
| Specialties/departments | ✅ Safe | Public information |
| NMC Registration Number | ✅ Safe | **Public registry** |
| Doctor's Name (with consent/public role) | ✅ Safe | Public professional |

### What is RISKY/NOT RECOMMENDED ⚠️
| Data Type | Risk Level | Reason |
|-----------|------------|--------|
| Doctor's personal mobile | ⚠️ Medium | Privacy concern |
| Doctor's home address | ❌ High | Personal safety/privacy |
| Doctor's personal email | ⚠️ Medium | Privacy concern |
| Patient data | ❌ Illegal | HIPAA/Privacy Act violation |
| Doctor's salary/income | ❌ High | Financial privacy |
| Unverified NMC numbers | ⚠️ Medium | Must verify from official source |

### Nepal Cyber Law Reference
- **Electronic Transaction Act 2063 (2008)**
- **Privacy Act 2018** (personal data protection)
- **Medical Council Act** (NMC numbers are public registry)

**VERDICT**: NMC numbers are PUBLIC records (like bar association numbers). Publishing verified NMC numbers from official NMC website is LEGAL. However, doctor personal contact info should be office/hospital numbers only.

## 🏥 CLINIC DATA STRUCTURE

### Safe & Legal Fields
```json
{
  "clinic_name": "City Heart Clinic",
  "type": "Cardiac Clinic",
  "address": "Main Road, Near Bus Park",
  "city": "Biratnagar",
  "district": "Morang",
  "province": "Koshi",
  "phone": ["021-460123", "021-460124"],
  "mobile": "9800000000",  // If publicly listed
  "email": "info@cityheart.com.np",
  "website": "https://cityheart.com.np",
  "operating_hours": "Sun-Fri: 9AM-5PM",
  "emergency_available": false,
  "services": ["ECG", "Echo", "Stress Test"],
  
  "doctors": [
    {
      "name": "Dr. Ram Sharma",
      "specialization": "Cardiologist",
      "nmc_number": "12345",  // ✅ Public registry
      "qualification": "MD, DM (Cardiology)",
      "available_days": "Sun, Tue, Thu",
      "available_time": "10AM-2PM"
      // NO personal mobile/email
    }
  ],
  
  "source": "nmc.org.np / clinic website",
  "last_verified": "2026-03-24"
}
```

### Data Verification Strategy
1. **Primary**: Official NMC website (nmc.org.np) - for NMC numbers
2. **Secondary**: Official district health office websites
3. **Tertiary**: Clinic/hospital official websites
4. **Avoid**: User-generated content, unverified directories

## ⚠️ RISK MITIGATION

### Data Quality Risks
| Risk | Mitigation |
|------|------------|
| Outdated phone numbers | Annual verification cycle |
| Incorrect NMC numbers | Only from nmc.org.np |
| Wrong locations | Google Maps API verification |
| Missing facilities | Crowdsource with verification |

### Legal Risks
| Risk | Mitigation |
|------|------------|
| Privacy complaints | Only public business info |
| Inaccurate doctor info | Source only from official NMC |
| Defamation | Accurate sourcing, correction process |
| Copyright | Original content, proper attribution |
